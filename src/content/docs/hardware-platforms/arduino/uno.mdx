# Arduino Uno

برد Arduino Uno یکی از محبوب‌ترین و پراستفاده‌ترین بردهای توسعه مبتنی بر میکروکنترلر در جهان است. این برد بر پایه میکروکنترلر ATmega328P ساخته شده و به دلیل سادگی، قیمت مناسب، منابع آموزشی گسترده و پشتیبانی عالی، به انتخاب اول بسیاری از علاقه‌مندان، دانشجویان و مهندسان الکترونیک تبدیل شده است.

Arduino Uno برای ساخت پروژه‌های آموزشی، رباتیک، اینترنت اشیاء (IoT)، اتوماسیون، کنترل سنسورها و صدها کاربرد دیگر استفاده می‌شود.

---

# نسخه‌های مختلف Arduino Uno

برد Arduino Uno معمولاً در دو نسخه اصلی عرضه می‌شود:

- Arduino Uno R3 DIP
- Arduino Uno R3 SMD

هر دو نسخه از نظر عملکرد تقریباً یکسان هستند اما در نوع بسته‌بندی میکروکنترلر تفاوت دارند.

---

# Arduino Uno R3 DIP

نسخه DIP از میکروکنترلر ATmega328P با پکیج DIP-28 استفاده می‌کند.

مزیت اصلی این مدل، قابل تعویض بودن میکروکنترلر است. در صورتی که تراشه آسیب ببیند یا نیاز به پروگرام جداگانه داشته باشید، می‌توانید به‌راحتی آن را از سوکت خارج کرده و تعویض کنید.

## ویژگی‌ها

- مناسب آموزش و یادگیری
- امکان تعویض آسان میکروکنترلر
- مناسب پروگرام مستقل AVR
- تعمیر آسان‌تر

## کاربردها

- پروژه‌های آموزشی
- نمونه‌سازی سریع
- تست و توسعه مدارها

---

# Arduino Uno R3 SMD

در نسخه SMD، میکروکنترلر ATmega328P به‌صورت سطحی (Surface Mount) روی برد لحیم شده است.

این نسخه ابعاد جمع‌وجورتری دارد و در تولید انبوه رایج‌تر است، اما میکروکنترلر آن به‌سادگی قابل تعویض نیست.

## ویژگی‌ها

- طراحی فشرده‌تر
- مقاومت بهتر در برابر لرزش
- مناسب تولید انبوه
- هزینه تولید پایین‌تر

## محدودیت‌ها

- تعویض دشوار میکروکنترلر
- تعمیر سخت‌تر نسبت به نسخه DIP

---

# تفاوت نسخه DIP و SMD

| ویژگی | DIP | SMD |
|---|---|---|
| نوع نصب تراشه | سوکتی | لحیم‌شده |
| امکان تعویض تراشه | دارد | ندارد |
| مناسب آموزش | بسیار مناسب | مناسب |
| مناسب تولید انبوه | متوسط | بسیار مناسب |
| تعمیرپذیری | آسان | دشوارتر |

---

# مشخصات فنی Arduino Uno

| مشخصه | مقدار |
|---|---|
| میکروکنترلر | ATmega328P |
| ولتاژ کاری | 5V |
| ولتاژ ورودی پیشنهادی | 7V تا 12V |
| تعداد پایه دیجیتال | 14 |
| تعداد PWM | 6 |
| تعداد ورودی آنالوگ | 6 |
| حافظه Flash | 32KB |
| SRAM | 2KB |
| EEPROM | 1KB |
| فرکانس کاری | 16MHz |

---

# میکروکنترلر ATmega328P

قلب اصلی Arduino Uno میکروکنترلر AVR مدل ATmega328P است.

این تراشه توسط شرکت Atmel (اکنون Microchip) تولید شده و شامل:

- CPU هشت‌بیتی AVR
- حافظه Flash
- تایمرها
- ADC
- PWM
- UART
- SPI
- I2C

می‌باشد.

این تراشه برای پروژه‌های سبک و متوسط بسیار مناسب است و مصرف توان پایینی دارد.

---

# پایه‌های دیجیتال

Arduino Uno دارای 14 پایه دیجیتال از D0 تا D13 است.

این پایه‌ها می‌توانند به‌عنوان:

- ورودی دیجیتال
- خروجی دیجیتال

استفاده شوند.

## پایه‌های PWM

پایه‌های زیر از PWM پشتیبانی می‌کنند:

- D3
- D5
- D6
- D9
- D10
- D11

این قابلیت برای:

- کنترل موتور
- تنظیم روشنایی LED
- تولید سیگنال

کاربرد دارد.

---

# پایه‌های آنالوگ

برد دارای 6 ورودی آنالوگ است:

- A0 تا A5

این پایه‌ها برای خواندن سنسورها و سیگنال‌های آنالوگ استفاده می‌شوند.

ADC داخلی Arduino Uno دقت 10 بیت دارد.

---

# ارتباطات سریال

Arduino Uno از چندین پروتکل ارتباطی پشتیبانی می‌کند.

## UART

پایه‌های:

- D0 (RX)
- D1 (TX)

برای ارتباط سریال استفاده می‌شوند.

---

## I2C

پایه‌های:

- A4 (SDA)
- A5 (SCL)

برای ارتباط I2C استفاده می‌شوند.

---

## SPI

پایه‌های:

- D10 (SS)
- D11 (MOSI)
- D12 (MISO)
- D13 (SCK)

برای SPI استفاده می‌شوند.

---

# تغذیه برد

برد Arduino Uno را می‌توان از روش‌های مختلف تغذیه کرد:

## USB

رایج‌ترین روش تغذیه و پروگرام برد.

---

## جک آداپتور DC

برد دارای جک تغذیه DC است.

ولتاژ پیشنهادی:

- 7V تا 12V

---

## پایه VIN

برای تغذیه مستقیم برد استفاده می‌شود.

---

# LEDهای روی برد

برد Arduino Uno چند LED داخلی دارد:

| LED | کاربرد |
|---|---|
| ON | نمایش روشن بودن برد |
| TX/RX | ارتباط سریال |
| L | متصل به پایه D13 |

LED داخلی D13 برای تست اولیه بسیار معروف است.

# پروگرام کردن برد

پروگرام Arduino Uno معمولاً با Arduino IDE انجام می‌شود.

مراحل کلی:

1. نصب Arduino IDE
2. اتصال برد با USB
3. انتخاب Board
4. انتخاب Port
5. Upload برنامه

# جمع‌بندی

Arduino Uno یکی از مهم‌ترین و تأثیرگذارترین بردهای توسعه در دنیای الکترونیک و سیستم‌های نهفته است. این برد با سادگی، انعطاف‌پذیری و اکوسیستم گسترده خود، همچنان یکی از بهترین گزینه‌ها برای آموزش، نمونه‌سازی و توسعه پروژه‌های الکترونیکی محسوب می‌شود.

نسخه DIP برای آموزش و توسعه انعطاف‌پذیر مناسب‌تر است، در حالی که نسخه SMD بیشتر برای استفاده صنعتی و تولید انبوه کاربرد دارد.
